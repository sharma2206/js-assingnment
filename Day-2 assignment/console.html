Various method of Console function

The console is useful for testing purposes.

1.The console.log() method writes a message to the console.
syntax: console.log(message)
2.The console.error() method writes an error message to the console.
syntax: console.error(message)
3.The console.clear() method will also write a message in the console: "Console was cleared"
synrax: console.clear()
4.The console.table() method writes a table in the console view
syntax: console.table(message)
5.The console.warn() method writes a warning to the console.
syntax: console.warn(message)
6.The console.info() method writes a message to the console.
syntax: console.warn(message)
